# Effective Java Notes

Concise, runnable notes and code examples from *Effective Java* (3rd Edition), validated with **SonarLint** for clean, production-ready practices.

## Overview
This repository contains practical demonstrations of *Effective Java* principles.  
Each item is supported by short, executable Java snippets focused on correctness, readability, and maintainability.

## Structure
- `src/main/java/` – Code snippets organized by *Effective Java* items  
- `notes/` – Explanatory markdown notes per chapter  
- `tests/` – Optional unit tests for validation

## Validation
All code is verified locally using **SonarLint** to ensure no code smells or static analysis issues remain.

## Requirements
- Java 17+  
- Maven or Gradle  
- IDE with SonarLint plugin (IntelliJ IDEA, VS Code, or Eclipse)
